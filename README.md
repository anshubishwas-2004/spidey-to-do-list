## ⚙️ Getting Started  

### 🔧 Prerequisites  

* Node.js (v14 or higher)

---

### 📥 Installation

```bash
git clone https://github.com/YOUR_USERNAME/spidey-todo.git
cd spidey-todo
npm install
```

### ▶️ Run the App

```bash
npm start
```

Now open 👉 **http://localhost:3000**

---

## 🧠 How It Works

1. User adds a task
2. Frontend sends request to the server
3. Express backend processes the task
4. Tasks are stored (memory / file / DB)
5. Updated list is sent back
6. UI updates instantly
